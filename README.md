### 📌 Projeto em Foco: API de Automação de DP
Uma API REST desenvolvida em Java com Spring Boot para automatizar rotinas de Departamento Pessoal. O sistema gerencia o cadastro de funcionários e gera folhas de pagamento, calculando automaticamente comissões, descontos e o salário líquido final. A aplicação utiliza Spring Data JPA para persistência de dados.
